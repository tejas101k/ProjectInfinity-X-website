Android 16 homecoming to Project Infinity X

Author: tejas101k • July 13, 2025

Finally the wait is over, Android 16 has arrived to Project Infinity X.. Project versioning will be v3.x from now on to mark it as successor version of v2.x (Android 15)

The source code for v3.0 is pushed at: https://github.com/ProjectInfinity-X and is ready to build

  • New QuickSettings Panel
  • Inherited aconfig release
  • New Animations
  • And much more from Baklava
